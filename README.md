## 🎮 Game Logic

- `7` → You (the player) 🧍‍♀️  
- `10` → Treasure 💎  
- `-1` → Mine 💣  
- `0` → Safe path ✅  
- 🕹️ Moves allowed: **5 only**  
- 🎯 Controls: `UP`, `DOWN`, `LEFT`, `RIGHT`
